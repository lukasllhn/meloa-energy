# meloa — Landing Page

Statische Single-File-Landing-Page (`index.html`, self-contained: Inline-CSS/JS, Google Fonts via CDN). Kein Build.

## Deploy (Vercel)
Repo importieren → Framework Preset **Other** → Deploy. `index.html` liegt im Root und wird unter `/` ausgeliefert.

## Lokal ansehen
`index.html` im Browser öffnen, oder im Ordner `python -m http.server` starten.
